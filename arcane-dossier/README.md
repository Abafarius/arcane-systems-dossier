# Arcane Systems Dossier

Static-first interactive portfolio for Galymzhan Yessimbek.

## Current checkpoint: Step 3

Implemented:

- Vite + React + TypeScript foundation
- Tailwind CSS v4 styling entry
- Static-first routing with `HashRouter`
- Design token CSS foundation
- Home page foundation
- Resume data and resume page
- Project case study system
- Project filters by category
- Individual project pages:
  - `#/projects/fincat`
  - `#/projects/ai-telegram-assistant`
  - `#/projects/learning-clicker`
  - `#/projects/cbdc-research`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
